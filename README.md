Running the on MATLAB SIMULINK 
1. Open Simulink and open the file ''
2. On MATLAB Terminal set the following variables:
      + f1=1
      + w1 = 2*pi*f1
      + R1=1000
      + R2=R1
      + C1=1/(w1*R1)
      + f2=10
      + w2 = 2*pi*f2 
      + C2 = 1/(2*pi*f2*R2)
      + G=2
